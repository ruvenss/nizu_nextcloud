Nizu NextCloud

What is Nizu NextCloud?

  • Nizu NextCloud allows your platform to sync information with the local installation of NextCloud via the SQL DataBase and the OCC Command.

Which are the features of Nizu NextCloud?

  • Sync the current members or clients contacts with the users of NextCloud
  • Sync the Groups

Which are the features are foreseen?

  • Sync files like Invoices, Members Files.
  • Sync Project Files
  • Sync Tasks Files
